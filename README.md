## 关于我

我喜欢把重复、零散、容易出错的流程整理成小而可靠的工具。当前重点在后端服务、桌面效率工具、Flutter 实验、原生设备接入、串口通信和自动化打包。

- 后端方向：Java / Rust / Go。
- 桌面工具：Swift / AppKit / macOS menu bar apps。
- 跨端与原生：Flutter / Dart / Android / C++ / serial ports。
- 工作偏好：清晰界面、稳定工具链、少步骤但能真正解决问题的项目。
